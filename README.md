# 프로젝트 실행방법
`npm start` 실행 

## 페이지 구현
1. path : '/'인 LoginAndRegister 컴포넌트
2. path : '/todo'인 Todos 컴포넌트
3. path : '*' 그외 처리인 NotFound 컴포넌트

## 배포링크
[링크](https://wanted-pre-onboarding-khw.vercel.app/)
